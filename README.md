# Security Assessment Lab

## Overview

This repository contains practical activities related to **cybersecurity and penetration testing**, performed on simulated environments (HackTheBox, TryHackMe).

The goal is to demonstrate:
- Technical skills in vulnerability assessment
- Structured penetration testing methodology
- Ability to produce professional security reports

---

## Structure

- **methodology/**  
  Standard penetration testing workflow  

- **tools/**  
  Custom scripts used during assessments  

- **reports/**  
  Professional reports (consulting-style)  

- **labs/**  
  Detailed technical notes and lab walkthroughs  

---

## Methodology

Activities follow a structured approach:

1. Reconnaissance  
2. Enumeration  
3. Exploitation  
4. Privilege Escalation  
5. Post-Exploitation  

---

## Tools

Commonly used tools include:
- Nmap
- LinPEAS
- gobuster
- FFUZ
- Hydra
- John

---

## Featured Assessments

### Custom Lab – Full Attack Chain
- Reverse engineering of binary (Ghidra)
- Credential extraction via XOR decoding
- Code injection via eval()
- Privilege escalation via rsync wildcard

Demonstrates advanced offensive security techniques across multiple layers.

## Disclaimer

All activities are conducted in **legal, controlled environments** for educational purposes only.
